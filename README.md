## 🚀 Cấu trúc trang web
Trang web được xây dựng với HTML, CSS, và Vanilla JavaScript, bao gồm các mục chính:
- **Trang chủ:** Giới thiệu tổng quan về nhóm SPRING và dự án.
- **Về chúng tôi:** Thông tin chi tiết, điểm mạnh, điểm yếu và vai trò của từng thành viên.
- **Dự án:** Thông tin về sản phẩm hệ thống bán vé sự kiện đang phát triển.
- **Tiến độ (Roadmap):** Theo dõi tiến độ công việc qua từng tuần (Khởi động, Phân tích thiết kế, Thực hiện đồ án, Báo cáo) và kho lưu trữ tài liệu PDF.
- **Thỏa thuận:** Các nguyên tắc làm việc chung, quy tắc thưởng phạt và bản hợp đồng nội bộ của nhóm.
